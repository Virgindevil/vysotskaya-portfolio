# 🎨 Сайт-портфолио художницы Высоцкой Добравы

> Персональный сайт-портфолио для демонстрации живописных работ, приёма заказов на комиссии и связи с клиентами.

## 🚀 Демо

- **Сайт**: [https://virgindevil.github.io/vysotskaya-portfolio](https://virgindevil.github.io/vysotskaya-portfolio)
- **Прототип в Figma**: [Ссылка на макет](https://www.figma.com/make/uijkUGluqo4l61WZElvOY9)

## 📦 Структура проекта
/ (root) 
├── index.html # Главная 
    
    ├── pages/ # Внутренние страницы 
    
    ├── css/ # Стили 
    
    ├── js/ # Скрипты 
    
    ├── img/ # Изображения (WebP)

## 🛠 Технологии

- **HTML5** (семантическая разметка, accessibility)
- **CSS3** (Flexbox, Grid, медиазапросы, анимации)
- **JavaScript** (ES6+, модули, валидация форм, модальные окна)
- **Swiper.js** (слайдер галереи)
- **WebP** (оптимизация изображений)

## 🧪 Тестирование

Проверено в:
- Chrome 120+, Firefox 121+, Safari 17+
- Мобильные: iOS Safari, Chrome Android
- Разрешения: 1440px, 1024px, 768px, 480px

## 📈 Оптимизация

- ✅ Изображения в WebP (~74% экономии)
- ✅ Lazy loading для изображений
- ✅ Минификация CSS/JS для production
- ✅ SEO meta-теги и структурированные данные
- ✅ PageSpeed: 92/100 (mobile), 98/100 (desktop)

## 🚀 Локальный запуск

```bash
# Простой способ (без сервера)
# Просто откройте index.html в браузере

# Через локальный сервер (рекомендуется для корректной работы путей)
# Python
python -m http.server 8000

# Node.js (если установлен live-server)
npx live-server

# Затем откройте: http://localhost:8000
```
